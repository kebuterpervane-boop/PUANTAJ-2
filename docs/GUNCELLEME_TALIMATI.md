# PuantajApp Güncelleme Talimatı

1. **Yeni Exe Dosyasını Alın**
   - Size gönderilen yeni `puantaj.exe` dosyasını indirin.

2. **Mevcut Klasöre Kopyalayın**
   - Eski programın bulunduğu klasöre yeni `puantaj.exe` dosyasını kopyalayın.
   - Eski `puantaj.exe` dosyasının üzerine yazabilirsiniz.
   - **Veritabanı dosyanıza (`puantaj.db`) dokunmayın!**

3. **Programı Açın**
   - Yeni exe’yi çalıştırın.
   - Program, eski veritabanınızı otomatik olarak yeni güvenli konuma taşıyacak ve yedekleyecektir.
   - Tüm verileriniz korunacaktır.

4. **Ekstra Bilgi**
   - Eğer installer ile kurulum yapıyorsanız, sadece yeni installer’ı çalıştırmanız yeterlidir.
   - Eski verileriniz yine otomatik olarak korunur.

**Sorun yaşarsanız eski klasördeki `puantaj.db` dosyanız yedeklenmiş olur, veri kaybı yaşamazsınız.**

Sorularınız için destek alabilirsiniz.
